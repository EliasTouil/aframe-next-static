## aframe-next-static

Boilerplate for building virtual reality experiences with
[A-Frame](https://aframe.io), React with
[aframe-react](https://github.com/ngokevin/aframe-react) and
[Next.js](https://github.com/zeit/next.js).

### Installation

To get started:

```bash
npm install
npm start
```

